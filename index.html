Bootstrap no-js

A group of Bootstrap components that work without JavaScript.

When using a progressive enhancement approach to building websites, we want our users to be able to access content without JavaScript as a requirement, then we can introduce JavaScript to enhance the experience if JavaScript is available.

To test these components be sure to disable JavaScript in your browser.

Getting Started

Each component is based off of Bootstrap's HTML but may require minimal markup changes and additional CSS to work without JavaScript. Read instructions for each component.

  1. Add a class of no-js to the html tag. e.g.
    <html class="no-js" lang="US-en">
  2. Include the following script inside the <head> of your project.
    <script>document.documentElement.className = document.documentElement.className.replace("no-js", "js");</script>
  3. Grab HTML/CSS from CodePen examples

Dropdowns

  1. Use <a> instead of <button> for toggle. This is so we can leverage :target for no-js scenario
  2. Add a unique ID to .dropdown wrapper eg. #custom-dropdown-id
  3. Add a unique ID to toggle e.g. #dropdown-toggle-id
  4. Add .nojs-only wrapper and a link back to the toggle. e.g. href="#dropdown-toggle-id"
  5. Remove aria-expanded="false" (JS would normally toggle this value)
  6. Include CSS

Suggestion: Use JavaScript to rewrite <a> toggle back into <button> for greater usability.

Modals

  1. Convert all <button> to <a>
  2. Add unique ID to toggle and .modal
  3. Point toggle to unique ID on href="#myModal"
  4. Point close buttons in .modal back to toggle. href="#unique-modal-trigger-id"
  5. Add backdrop inside .modal e.g.
    <div class="modal-backdrop fade in nojs-only"></div>
  6. Include CSS

Suggestion: Use JavaScript to rewrite <a> toggles back into <button> for greater usability.
