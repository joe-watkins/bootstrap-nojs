Bootstrap no-js

A group of Bootstrap components that work without JavaScript.

Below you will find some components from the Bootstrap framework that work without JavaScript. To test these components be sure to disable JavaScript in your browser. A few HTML modifications may be needed to make each component work.

Getting Started

Each component is based off of Bootstrap's HTML but may require minimal markup changes and additional CSS to work without JavaScript. Read instructions for each component.

  1. Add a class of no-js to the html tag. e.g. <html class="no-js" lang="US-en">
  2. Include the following script inside the <head> of your project.
    <script>document.documentElement.className = document.documentElement.className.replace("no-js", "js");</script>
  3. Grab HTML/CSS from CodePen examples

Dropdowns

  1. Use <a> instead of <button> for toggle. This is so we can leverage :target for no-js scenario
  2. Add a unique ID to .dropdown wrapper eg. #custom-dropdown-id
  3. Add a unique ID to toggle e.g. #dropdown-toggle-id
  4. Add .nojs-only wrapper and a link back to the toggle. e.g. href="#dropdown-toggle-id"
  5. Remove aria-expanded="false" (JS would normally toggle this value)
  6. Include CSS
